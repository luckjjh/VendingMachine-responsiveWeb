### 배우게 된 부분
* html tag를 작성할 때 이후 적용할 CSS, JavaScript를 고려하게 됨
* CSS를 적용하기 전에 공통적인 스타일을 갖는 요소들을 분류해 하나의 class로 통일 시켜 css 코드를 줄여야 한다는 생각을 갖게 됨
* grid, flex, float을 활용해 레이아웃을 유연하게 구성하게 됨
